Subjects
  • Advanced Software Engineering(ASE)

    Advanced Software Engineering covers about the different steps that we have to follow in order to develop an efficient,reliable and quality software.
    Section includes Life Cycle od Software Engineering,features of SE etc.

  • Advances Data Structures(ADS)

    Advanced Data Structures covers the basic six operations insert,delete,sort,search,traverse.
    Various techniques like binary search,tree,linear sort etc are discussed.

  • Python Lab(PL)

    Python Programming Lab covers the basics of Python.
    Basic input output syntax ,loops,if,files etc are covered here.

  • Web Programming Lab(WPL)

    Web Programming Lab course teaches you on how to create a web page and handle its frond end and back end.
    It covers html,javascript,PHP.

  • Advanced Data Structures Lab(ADSL)

    Advanced Data Structures deal with different type of data structures.
    These data structure include link list,tree,queue etc.

  • Mathematics(M)

    It discuss basics of mathematics.
    It covers probability,matrices,sets etc

  • Digital Fundamentals(DF)

    The subjects covers the fundamental structure of computer.
    It discuss about logical gates,structures etc.
